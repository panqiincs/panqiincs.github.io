2017年的几个小目标

发表于 2017-01-03 | 分类于 生活记录 | 评论数：

2016年是非常重要的一年，但到现在还没写一篇总结，后面一定要补上。去年7月底参加工作之后，属于自己的时间骤减，这造成了如下的后果：

  1. 阅读量骤减：2016年一共只读了13本书，参加工作之后只完成了一本书
  2. 技术学习减少：业余时间的技术学习一度中断，很担心自己的技术只局限在工作相关的领域
  3. 身体不舒服：腰酸背疼的，前几天往秤上一站，发现自己竟然胖了4斤

每年都应该有计划，2017年要从这几个方面改进，定下几个小目标：

  1. 阅读：读10本非技术书籍，至少完成《经济学原理》、《影响力》、《如何阅读一本书》
  2. 技术学习：至少读完《TCP/IP详解》、《重构》，APUE再通读一遍；完成一个shell；完成一个http server；学会写Python爬虫；写有价值的文章，博客日均访问量突破35
  3. 锻炼：体重保持在正常范围内；5公里跑进25分钟；加强上肢力量锻炼

当然，这一切的前提是：提升工作效率，减少加班。

Shellе®һзҺ°пјҡеҹәжң¬еҠҹиғҪ

еҸ‘иЎЁдәҺ 2017-02-26 еҲҶзұ»дәҺ зЁӢеәҸи®ҫи®Ў Disqusпјҡ

Shellзҡ„еҠҹиғҪ

ShellжҳҜж“ҚдҪңзі»з»ҹдёӯз®ЎзҗҶиҝӣзЁӢе’ҢиҝҗиЎҢзЁӢеәҸзҡ„зЁӢеәҸгҖӮжүҖжңүеёёз”Ёзҡ„shellйғҪжңүдёүдёӘдё»иҰҒеҠҹиғҪпјҡ

  • иҝҗиЎҢзЁӢеәҸпјҡgrep, date, lsзӯүйғҪжҳҜдёҖдәӣжҷ®йҖҡзҡ„зЁӢеәҸпјҢshellе°Ҷе®ғд»¬иҪҪе…ҘеҶ…еӯҳе№¶иҝҗиЎҢе®ғд»¬
  • з®ЎзҗҶиҫ“е…Ҙе’Ңиҫ“еҮәпјҡдҪҝз”Ё<пјҢ>е’Ң | з¬ҰеҸ·еҸҜд»Ҙе°Ҷиҫ“е…ҘгҖҒиҫ“еҮәйҮҚе®ҡеҗ‘гҖӮиҝҷж ·е°ұеҸҜд»Ҙе‘ҠиҜүshellе°ҶиҝӣзЁӢзҡ„иҫ“е…Ҙе’Ңиҫ“еҮәиҝһжҺҘеҲ°дёҖдёӘж–Үд»¶жҲ–жҳҜе…¶д»–зҡ„иҝӣзЁӢ
  • еҸҜзј–зЁӢпјҡshellеҗҢж—¶д№ҹжҳҜеёҰжңүеҸҳйҮҸе’ҢжөҒзЁӢжҺ§еҲ¶зҡ„зј–зЁӢиҜӯиЁҖ

жң¬ж–Үе°Ҷд»Ӣз»ҚеҰӮдҪ•зј–еҶҷеҢ…еҗ«жңҖеҹәжң¬еҠҹиғҪпјҢеҚіиғҪиҝҗиЎҢе’Ңз»ҲжӯўзЁӢеәҸзҡ„shellгҖӮ

Shellзҡ„дё»жөҒзЁӢ

Shellжү“еҚ°жҸҗзӨәз¬ҰгҖҒиҫ“е…Ҙе‘Ҫд»ӨгҖҒиҝҗиЎҢе‘Ҫд»ӨпјҢеҰӮжӯӨеҸҚеӨҚгҖӮеӣ жӯӨпјҢдё»жөҒзЁӢз”ұдёӢйқўзҡ„еҫӘзҺҜз»„жҲҗпјҡ

жү“еҚ°жҸҗзӨәз¬Ұ

дёӢйқўжҳҜдёҖдёӘжҸҗзӨәз¬Ұзҡ„зӨәдҫӢпјҡ

еҠ дёҠеүҚзјҖ[psh]д»Ҙе’Ңзі»з»ҹиҮӘеёҰзҡ„shellеҢәеҲ«ејҖжқҘгҖӮеҗҺйқўдҫқж¬ЎжҳҜз”ЁжҲ·еҗҚпјҲз”Ёgetlogin_rеҮҪж•°иҺ·еҸ–пјүгҖҒдё»жңәеҗҚпјҲз”ЁgethostnameеҮҪж•°иҺ·еҸ–пјүгҖҒеҪ“еүҚзӣ®еҪ•пјҲз”ЁgetcwdеҮҪж•°иҺ·еҸ–пјүе’Ңе‘Ҫд»ӨжҸҗзӨәз¬ҰгҖӮеҰӮжһңеҪ“еүҚзҷ»еҪ•зҡ„з”ЁжҲ·жҳҜrootпјҢе‘Ҫд»ӨжҸҗзӨәз¬Ұдёә#пјҢжҷ®йҖҡз”ЁжҲ·еҲҷжү“еҚ°зҫҺе…ғз¬ҰеҸ·$пјҢеӣ жӯӨйңҖиҰҒз”ЁgeteuidеҮҪж•°иҺ·еҸ–еҪ“еүҚзҡ„Effective User IDгҖӮ

еӨ„зҗҶз”ЁжҲ·иҫ“е…Ҙ

дёҖиҲ¬йңҖиҰҒйҷҗеҲ¶з”ЁжҲ·еҚ•ж¬Ўиҫ“е…Ҙзҡ„жңҖеӨҡеӯ—з¬Ұж•°MAX_ARG_LENд»ҘеҸҠеҸӮж•°зҡ„жңҖеӨ§дёӘж•°MAX_ARG_NUMгҖӮз”ЁжҲ·иҫ“е…Ҙе‘Ҫд»ӨпјҢжҢүдёӢеӣһиҪҰй”®еҚіиЎЁзӨәеҪ“еүҚзҡ„е‘Ҫд»Өиҫ“е…Ҙе®ҢжҜ•гҖӮз”ЁgetlineеҮҪж•°иҜ»еҸ–дёҖиЎҢз”ЁжҲ·иҫ“е…ҘпјҢз„¶еҗҺе°ҶдёҖж•ҙжқЎе‘Ҫд»ӨжӢҶеҲҶгҖӮдҫӢеҰӮе‘Ҫд»Өls -al /homeжӢҶеҲҶд№ӢеҗҺеҸҳжҲҗlsпјҢ-alе’Ң/homeдёүдёӘеӯ—з¬ҰдёІпјҢз¬¬дёҖдёӘеӯ—з¬ҰдёІжҳҜе‘Ҫд»Өзҡ„еҗҚз§°пјҢеҗҺйқўдҫқж¬ЎжҳҜе‘Ҫд»ӨеҸӮж•°гҖӮжӢҶеҲҶеӯ—з¬ҰдҪҝз”ЁstrsepеҮҪж•°пјҢеҲҶеүІз¬ҰпјҲdelimeterпјүдёәз©әж јпјҲSpaceпјүгҖӮдёӢйқўжҳҜжӢҶеҲҶиҝҮзЁӢзҡ„д»Јз Ғпјҡ

е…¶дёӯinput_bufжҳҜдёҖдёӘеӯ—з¬Ұж•°з»„пјҢеӯҳж”ҫз”ЁжҲ·иҫ“е…Ҙзҡ„еҚ•жқЎе‘Ҫд»Өеӯ—з¬ҰдёІпјҢarg_vecжҳҜеӯ—з¬ҰдёІж•°з»„пјҢеӯҳж”ҫжӢҶеҲҶеҗҺзҡ„еӨҡдёӘеӯ—з¬ҰдёІгҖӮ

иҝҗиЎҢе‘Ҫд»Ө

forkе’Ңexec

жғіеҝ…еӨ§е®¶йғҪзҶҹжӮүshellиҝҗиЎҢе‘Ҫд»Өзҡ„жөҒзЁӢпјҡз”ЁжҲ·иҫ“е…ҘдёҖжқЎе‘Ҫд»ӨеҗҺпјҢзҲ¶иҝӣзЁӢдјҡforkдёҖдёӘеӯҗиҝӣзЁӢпјҢеңЁеӯҗиҝӣзЁӢдёӯдҪҝз”ЁexecеҮҪж•°иҝҗиЎҢз”ЁжҲ·иҫ“е…Ҙзҡ„е‘Ҫд»ӨпјҢзҲ¶иҝӣзЁӢзӯүеҫ…еӯҗиҝӣзЁӢйҖҖеҮәеҗҺпјҢзӯүеҫ…з”ЁжҲ·иҫ“е…ҘдёӢдёҖжқЎе‘Ҫд»ӨпјҢеҰӮжӯӨеҸҚеӨҚгҖӮд»Јз ҒеҰӮдёӢпјҡ

execеҮҪж•°жҜ«ж— з–‘й—®йҖүжӢ©жңҖж–№дҫҝзҡ„execvpпјҢдё»иҝӣзЁӢдҪҝз”ЁwaitеҮҪж•°зӯүеҫ…еӯҗиҝӣзЁӢйҖҖеҮәгҖӮ

еҶ…е»әе‘Ҫд»Ө

иҝҷдёӘж—¶еҖҷдёҖдёӘеҸҜд»Ҙи·‘зҡ„shellе°ұе®ҢжҲҗдәҶгҖӮдҪҶжҳҜеҚҙж— жі•иҝҗиЎҢcdе‘Ҫд»ӨпјҢжҸҗзӨәй”ҷиҜҜеҰӮдёӢпјҡ

иҝҗиЎҢwhereis cdе‘Ҫд»ӨпјҢе№¶жІЎжңүжҳҫзӨәеҸҜжү§иЎҢж–Үд»¶зҡ„и·Ҝеҫ„гҖӮеҺҹжқҘcdе‘Ҫд»ӨжҳҜеҶ…е»әе‘Ҫд»ӨпјҲbuild in commandпјүпјҢйңҖиҰҒеңЁshellдёӯиҮӘе·ұе®һзҺ°пјҢиҖҢдё”е®ғзҡ„жү§иЎҢдёҚйңҖиҰҒе»әз«ӢеӯҗиҝӣзЁӢпјҢйңҖиҰҒйўқеӨ–еҲӨж–ӯе’ҢеӨ„зҗҶгҖӮ

жҲ‘еҶҷдәҶдёҖдёӘеҮҪж•°еҲӨж–ӯжҳҜеҗҰжҳҜеҶ…е»әе‘Ҫд»ӨпјҢеҰӮжһңдёҚжҳҜеҶ…е»әе‘Ҫд»ӨеҲҷиҝ”еӣһ-1пјҢеҗҰеҲҷжҢүз…§еҶ…е»әе‘Ҫд»Өзҡ„ж–№ејҸиҝҗиЎҢгҖӮзӣ®еүҚеҸӘе®һзҺ°дәҶcdе’ҢexitдёӨдёӘеҶ…е»әе‘Ҫд»ӨгҖӮд»Јз ҒеҰӮдёӢпјҡ

жӯӨж—¶пјҢshellдё»еҫӘзҺҜдёӯзҡ„run_cmdеҮҪж•°жј”еҸҳдёәпјҡ

еӨ„зҗҶдҝЎеҸ·

жҲ‘д»¬жңҹжңӣshellиғҪеӨҹеӨ„зҗҶдҝЎеҸ·гҖӮеңЁshellзҡ„е‘Ҫд»ӨиЎҢдёӯж•Іе‘Ҫд»ӨиҝҗиЎҢзЁӢеәҸпјҢеңЁзЁӢеәҸиҝҳеңЁиҝҗиЎҢж—¶жҢүдёӢCtrl-CжҲ–Ctrl-\й”®пјҲеҲҶеҲ«дә§з”ҹSIGINTе’ҢSIGQUITдҝЎеҸ·пјүпјҢиҝҗиЎҢзҡ„зЁӢеәҸдјҡйҖҖеҮәпјҢеҚідёӯж–ӯиҝҗиЎҢеӯҗиҝӣзЁӢпјӣеңЁshellзҡ„е‘Ҫд»ӨиЎҢдёӯдёҚиҝҗиЎҢе‘Ҫд»ӨпјҢзӣҙжҺҘжҢүдёӢиҝҷдёӨдёӘй”®з»„еҗҲпјҢдёҚиғҪйҖҖеҮәshellзЁӢеәҸпјҢеҚізҲ¶иҝӣзЁӢдёҚдјҡдёӯж–ӯиҝҗиЎҢпјҢйҖҖеҮәshellзЁӢеәҸеҸӘиғҪйҖҡиҝҮCtrl-DжҢүй”®гҖӮе®һзҺ°дёҠиҝ°еҠҹиғҪйңҖиҰҒеңЁзҲ¶иҝӣзЁӢдёӯеҝҪз•ҘдҝЎеҸ·SIGINTе’ҢSIGQUITпјҢдҪҶжҳҜеңЁеӯҗиҝӣзЁӢдёӯжҒўеӨҚеҜ№дҝЎеҸ·SIGINTе’ҢSIGQUITзҡ„й»ҳи®Өж“ҚдҪңгҖӮд»Јз ҒеҫҲз®ҖеҚ•пјҢеңЁиҝҷйҮҢдёҚеҲ—еҮәгҖӮ

иҝҷж ·пјҢдёҖдёӘе…·еӨҮеҹәжң¬еҠҹиғҪзҡ„shellе°ұе®ҢжҲҗдәҶпјҢе®Ңж•ҙзҡ„д»Јз ҒиҜ·еҸӮиҖғиҝҷйҮҢгҖӮиҜ·з»§з»ӯзңӢдёӢдёҖзҜҮпјҡShellе®һзҺ°пјҡйҮҚе®ҡеҗ‘е’Ңз®ЎйҒ“гҖӮ

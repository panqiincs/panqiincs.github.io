童年的小龙虾

发表于 2019-05-31 分类于 生活记录 Valine：

最近这几年，小龙虾成为网红食物。油焖大虾名声在外，引得食客蜂拥而至。但是在我眼里，油焖大虾哪比得上我小时候自己钓、自己洗、自己做的小龙虾呢？

在我的家乡，一到春夏之交，小水塘里随处可见青色的、红色的小龙虾。钓龙虾的工具非常简单，找根一米多长的树枝绑上棉线就是钓竿，棉线另一端系上饵料。我一般用容易找到的黑色大蚯蚓做饵料。最好还准备一个抄网，我和小伙伴们一般自己用铁丝和渔网制作。还需要提个桶，用来装钓上来的龙虾。

钓小龙虾可比钓鱼有意思多了。龙虾喜欢在岸边的水草附近活动，发现它的行踪后，将诱饵垂到它附近的水面，它发现诱饵后就用钳子去夹。青色的是年轻的龙虾，虾生经验不足，碰到好吃的就夹得紧紧的，这时你抬起吊杆，把它提出水面，悬在空中，它的钳子都不会松开，可以直接转移到桶里。大红色的是身经百战的龙虾，见得多了，知道有人想钓它，非常警惕，你一抬杆或一将它提出水面它就松开钳子，很难钓上来。这时候，抄网就大显身手了。一手持钓竿，一手拿抄网，等龙虾夹住饵料尽情享用时，把抄网偷偷伸入水中，直到龙虾的底部，猛地往上一捞，它就落网了。

提着钓到的半桶龙虾回家，接下来是清洗工作。我会掰掉前面的壳，去掉虾黄，拔掉黑色的虾线，用牙刷仔细把脏东西刷掉，最后用清水冲洗干净。把小龙虾清洗干净很耗费时间，现在的酒店可就省事多了，龙虾身上有很多黑色的残留物。我们烹饪小龙虾的方法很简单：烧热油，大蒜下锅炒香，倒入小龙虾翻炒，加点酱油、醋、辣酱，炒熟了就可以出锅。我童年的生活并不富足，大鱼大肉不是随便就能吃到的，所以小龙虾是美味佳肴，对大人来说还是下酒的好菜。

二十年前的中国比较贫困，在那种条件下，农村的小孩想办法找好玩的好吃的，自己动手寻找生活的乐趣。所以我童年有很多美好的回忆，钓龙虾就是其中之一。六一儿童节马上就到了，祝小朋友们都能拥有幸福快乐的童年。

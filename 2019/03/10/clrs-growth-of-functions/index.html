гҖҺз®—жі•еҜји®әгҖҸз¬¬ 3 з« пјҡеҮҪж•°зҡ„еўһй•ҝ

еҸ‘иЎЁдәҺ 2019-03-10 | еҲҶзұ»дәҺ ж•°жҚ®з»“жһ„ | иҜ„и®әж•°пјҡ

дё»иҰҒеҶ…е®№

жң¬з« и®Іи§ЈеҰӮдҪ•з®ҖеҢ–з®—жі•зҡ„жёҗиҝ‘еҲҶжһҗпјҢд»Ӣз»ҚдәҶеҮ зұ»жёҗиҝ‘и®°еҸ·гҖӮжңҖеҗҺз®ҖеҚ•д»Ӣз»ҚдәҶеёёз”ЁеҮҪж•°зҡ„жҖ§иҙЁгҖӮжң¬з« зҡ„еҶ…е®№еңЁиҝҷйҮҢжҸҸиҝ°еҫ—йқһеёёз®Җз•ҘпјҢдҪҶдёәдәҶдҝқжҢҒж•ҙдёӘзі»еҲ—зҡ„е®Ңж•ҙжҖ§пјҢиҝҳжҳҜиҙҙеҮәжқҘдәҶгҖӮ

жёҗиҝ‘и®°еҸ·

жң¬з« д»Ӣз»ҚдәҶеҰӮдёӢжёҗиҝ‘и®°еҸ·пјҢе®ғд»¬зҡ„дёҘж је®ҡд№үеҲҶеҲ«еҰӮдёӢпјҡ

  1. иӢҘ$f(n) = \Theta(g(n))$пјҢеҲҷеӯҳеңЁжӯЈеёёйҮҸ$c_1$гҖҒ$c_2$е’Ң$n_0$пјҢдҪҝеҫ—еҜ№жүҖжңү$n \ge n_0$пјҢжңү$0 \le c_1 g(n) \le f(n) \le c_2 g(n)$
  2. иӢҘ$f(n) = O(g(n))$пјҢеҲҷеӯҳеңЁжӯЈеёёйҮҸ$c$е’Ң$n_0$пјҢдҪҝеҫ—еҜ№жүҖжңү$n \ge n_0$пјҢжңү$0 \le f(n) \le c g(n)$
  3. иӢҘ$f(n) = \Omega(g(n))$пјҢеҲҷеӯҳеңЁжӯЈеёёйҮҸ$c$е’Ң$n_0$пјҢдҪҝеҫ—еҜ№жүҖжңү$n \ge n_0$пјҢжңү$0 \le c g(n) \le f(n)$
  4. иӢҘ$f(n) = o(g(n))$пјҢеҲҷеҜ№д»»ж„Ҹ$c>0$пјҢеӯҳеңЁеёёйҮҸ$n_0>0$пјҢдҪҝеҫ—еҜ№жүҖжңү$n \ge n_0$пјҢжңү$0 \le f(n) < c g(n)$
  5. иӢҘ$f(n) = \omega(g(n))$пјҢеҲҷеҜ№д»»ж„Ҹ$c>0$пјҢеӯҳеңЁеёёйҮҸ$n_0>0$пјҢдҪҝеҫ—еҜ№жүҖжңү$n \ge n_0$пјҢжңү$0 \le c g(n) < f(n)$

дёәдәҶжӣҙеҪўиұЎең°зҗҶи§Јжң¬иҠӮд»Ӣз»Қзҡ„еҮ з§Қжёҗиҝ‘и®°еҸ·пјҢеҸҜд»ҘеңЁдёӨдёӘеҮҪж•°зҡ„жёҗиҝ‘жҜ”иҫғе’ҢдёӨдёӘе®һж•°зҡ„жҜ”иҫғд№Ӣй—ҙеҒҡдёҖз§Қзұ»жҜ”пјҡ

  • $f(n) = O(g(n))$зұ»дјјдәҺ$a \le b$
  • $f(n) = \Omega(g(n))$зұ»дјјдәҺ$a \ge b$
  • $f(n) = \Theta(g(n))$зұ»дјјдәҺ$a = b$
  • $f(n) = o(g(n))$зұ»дјјдәҺ$a < b$
  • $f(n) = \omega(g(n))$зұ»дјјдәҺ$a > b$

ж ҮеҮҶи®°еҸ·дёҺеёёз”ЁеҮҪж•°

жң¬иҠӮдё»иҰҒи®Іи§ЈдәҶеёёз”Ёзҡ„ж•°еӯҰеҮҪж•°е’Ңи®°еҸ·пјҢж¶үеҸҠеҲ°й«ҳдёӯж•°еӯҰе’ҢеҹәзЎҖеҫ®з§ҜеҲҶзҡ„зҹҘиҜҶгҖӮ

д№ йўҳи§Јзӯ”

з»ғд№ йўҳ

3.1-1 и§Јзӯ”пјҡеҚійңҖиҜҒжҳҺеӯҳеңЁжӯЈеёёж•°$c_1$пјҢ$c_2$е’Ң$n_0$пјҢдҪҝеҫ—еҪ“$n \ge n_0$ж—¶жңүпјҡ

$$ c_1(f(n)+g(n)) \le \max(f(n), g(n)) \le c_2(f(n)+g(n)) $$

з”ұдәҺ$f(n)$е’Ң$g(n)$еқҮжҳҜжёҗиҝ‘йқһиҙҹеҮҪж•°пјҢжүҖд»ҘеӯҳеңЁ$n_0$пјҢдҪҝеҫ—еҪ“$n \ge n_0$ж—¶пјҢжңү$f(n) \ge 0$е’Ң$g(n) \ge 0$гҖӮд»Ө

$$ h(x) = \max(f(n), g(n))$$

еҲҷ

$$ 0 \le f(n) \le h(n) \\
0 \le g(n) \le h(n) $$

жүҖд»Ҙжңү

$$ 0 \le f(n)+g(n) \le 2h(n) \\
0 \le (f(n)+g(n))/2 \le h(n) $$

еҸҲжңү

$$ h(n) \le f(n)+g(n) $$

жүҖд»ҘеӯҳеңЁ$c_1 = 1/2$пјҢ$c_2 = 1$е’Ң$n_0 \ge 0$ж»Ўи¶іжқЎд»¶пјҢеҫ—иҜҒгҖӮ

3.1-2 и§Јзӯ”пјҡеҚійңҖиҜҒжҳҺеӯҳеңЁжӯЈеёёж•°$c_1$пјҢ$c_2$е’Ң$n_0$пјҢдҪҝеҫ—еҪ“$n \ge n_0$ж—¶жңүпјҡ

$$ c_1 n^b \le (n+a)^b \le c_2 n^b $$

еҚі

$$ c_1 \le (1+a/n)^b \le c_2 $$

еҪ“$b > 0$ж—¶пјҢе№ӮеҮҪж•°еңЁ$[0, +\infty)$дёҠеҚ•и°ғйҖ’еўһгҖӮдёӢйқўж №жҚ®$a$зҡ„з¬ҰеҸ·еҲҶжғ…еҶөи®Ёи®әпјҡ

  1. еҪ“$a > 0$ж—¶пјҢд»Ө$n \ge n_0 = a$пјҢеҲҷжңү$1 \le 1+a/n \le 2$пјҢеҚі$c_1 = 1, c_2 = 2$пјҢж №жҚ®е№ӮеҮҪж•°зҡ„еҚ•и°ғжҖ§пјҢдёҠејҸжҲҗз«Ӣпјӣ
  2. еҪ“$a < 0$ж—¶пјҢ$1+a/n = 1-|a|/n$пјҢд»Ө$n \ge n_0 = 2|a|$пјҢеҲҷжңү$1/2 \le (1-|a|/n) \le 1$пјҢеҚі$c_1 = 1/2, c_2 = 1$пјҢж №жҚ®е№ӮеҮҪж•°зҡ„еҚ•и°ғжҖ§пјҢдёҠејҸжҲҗз«ӢгҖӮ

жүҖд»Ҙеҫ—иҜҒгҖӮ

3.1-3 и§Јзӯ”пјҡеӣ дёә$O$з¬ҰеҸ·жІЎжңүжҸҗдҫӣдёҖдёӘдёӢз•ҢгҖӮ

3.1-4 и§Јзӯ”пјҡ$2^{n+1} = O(2^n)$жҲҗз«ӢпјҢ$2^{2n} = O(2^n)$дёҚжҲҗз«ӢгҖӮ

3.1-5 и§Јзӯ”пјҡиҰҒиҜҒжҳҺзҡ„е…¬ејҸжҳҜпјҡ

$$ \lim_{n \to \infty}f(n)/g(n) = 0 $$

ж №жҚ®ж•°еҲ—жһҒйҷҗзҡ„$\epsilon-N$е®ҡд№үпјҢеҜ№д»»ж„Ҹ$\epsilon > 0$пјҢеӯҳеңЁиҮӘз„¶ж•°$N$дҪҝеҫ—еҪ“$n > N$ж—¶пјҢжңү$|f(n)/g(n)| < \epsilon$гҖӮ

ж №жҚ®$f(n) = o(g(n))$зҡ„е®ҡд№үпјҢеҜ№д»»ж„ҸжӯЈеёёйҮҸ$c > 0$пјҢеӯҳеңЁ$n_0 > 0$пјҢдҪҝеҫ—еҜ№жүҖжңү$n \ge n_0$пјҢжңү$0 \le f(n) < cg(n)$пјҢеҚі$|f(n)/g(n)| < c$пјҢеҸ–$\epsilon = c$пјҢ$N = n_0$пјҢеҫ—иҜҒгҖӮ

3.1-6 и§Јзӯ”пјҡ$O$и®°еҸ·жҸҗдҫӣдәҶдёҖдёӘжёҗиҝ‘дёҠз•ҢпјҢ$\Omega$и®°еҸ·жҸҗдҫӣдәҶжёҗиҝ‘дёӢз•ҢпјҢжүҖд»Ҙдёә$\Theta(g(n))$гҖӮдёҘж јиҜҒжҳҺз•ҘгҖӮ

3.1-7 и§Јзӯ”пјҡзұ»дјјдәҺ$a < b$е’Ң$a > b$дёҚеҸҜиғҪеҗҢж—¶жҲҗз«ӢгҖӮ

3.1-8 и§Јзӯ”пјҡеҰӮдёӢпјҡ

$f(n, m) = \Omega(g(n, m))$пјҢеҲҷеӯҳеңЁжӯЈеёёйҮҸ$c$, $n_0$е’Ң$m_0$пјҢдҪҝеҫ—еҜ№жүҖжңүзҡ„$n \ge n_0$жҲ–$m \ge m_0$пјҢжңү$0 \le cg(n, m) \le f(n, m)$.
$f(n, m) = \Theta(g(n, m))$пјҢеҲҷеӯҳеңЁжӯЈеёёйҮҸ$c_1$пјҢ$c_2$пјҢ$n_0$е’Ң$m_0$пјҢдҪҝеҫ—еҜ№жүҖжңүзҡ„$n \ge n_0$жҲ–$m \ge m_0$пјҢжңү $c_1 g(n, m) \le f(n, m) \le c_2 g(n, m)$.

3.2-1 и§Јзӯ”пјҡжҢүз…§еҚ•и°ғжҖ§зҡ„е®ҡд№үеҚіеҸҜиҜҒжҳҺпјҢз•ҘгҖӮ

3.2-2 и§Јзӯ”пјҡз”ұе…¬ејҸ$a = b^{\log_b a}$еҫ—пјҡ

$$
a^{\log_b c} = (b^{\log_b a})^{\log_b c} = b^{\log_b a \log_b c} \\
c^{\log_b a} = (b^{\log_b c})^{\log_b a} = b^{\log_b c \log_b a}
$$

еҫ—иҜҒгҖӮ

3.2-3 и§Јзӯ”пјҡиҰҒиҜҒжҳҺзҡ„(3.19)ејҸдёә$n! = \Theta(n \lg n)$пјҢеҖҹеҠ©ж–Ҝзү№жһ—иҝ‘дјје…¬ејҸпјҢиҜҒжҳҺеҰӮдёӢпјҡ

$$
\lg(n!) = \lg \sqrt{2 \pi n} + \lg (n/e)^n + \lg (1+\Theta(1/n)) = \Theta(\lg n) + \Theta(n \lg n) = \Theta(n \lg n)
$$

иҰҒиҜҒжҳҺ$n! = o(n^n)$пјҢиҜҒжҳҺ$\lim_{n \to \infty}n!/n^n = 0$еҚіеҸҜпјҢд»Је…Ҙж–Ҝзү№жһ—иҝ‘дјје…¬ејҸеҚіеҫ—иҜҒгҖӮ

иҰҒиҜҒжҳҺ$n! = \omega(2^n)$пјҢиҜҒжҳҺ$\lim_{n \to \infty}n!/2^n = \infty$еҚіеҸҜгҖӮд»Ө$h(n) = n!/2^n$пјҢеҲҷпјҡ

$$h(n) = \frac{n!}{2^n} = \frac{(n-1)!}{2^{n-1}} \frac{n}{2} = h(n-1) \frac{n}{2}$$

еҪ“$n=4$ж—¶пјҢ$h(n)>1$пјҢеҪ“$n>4$ж—¶пјҢ$n/2 > 2$пјҢжүҖд»Ҙ$h(n) > 2^{n-4}$пјҢжүҖд»Ҙ$h(n)$зҡ„жһҒйҷҗдёә$\infty$гҖӮ

3.2-6 и§Јзӯ”пјҡж №жҚ®й»„йҮ‘еҲҶеүІзҺҮзҡ„е®ҡд№үжңүпјҡ$\phi / 1 = \phi / (1 - \phi)$пјҢеҚіж»Ўи¶іж–№зЁӢ$x^2 = x + 1$пјҢеҫ—иҜҒгҖӮ

3.2-7 и§Јзӯ”пјҡеҪ’зәіжі•зҡ„жӯҘйӘӨеҰӮдёӢпјҡ

  1. еҲқе§Ӣжғ…еҶө$i = 0$ж—¶пјҢ$F_0 = (\phi^0 - \hat{\phi^0})/\sqrt{5} = 0$пјҢ$i = 1$ж—¶пјҢ$F_1 = (\phi - \hat{\phi})/\sqrt{5} = 1$пјҢз¬ҰеҗҲ

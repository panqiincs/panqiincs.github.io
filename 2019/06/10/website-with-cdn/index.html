博客用上了CDN

发表于 2019-06-10 | 分类于 网站维护 | 评论数：

我的博客托管在Vultr的虚拟主机上，稳定运行快两年了。最近Vultr的日本机房全线崩溃，于是我又忙着把博客迁回到Github上。托管在Github上，丧失了自主性，但省却了维护的麻烦，还节约了一笔费用。以前还有换动态框架的念头，经历此事后，唉，能简单就简单。

看到有个独立博客也有类似的经历，他还用上了Cloudflare的免费CDN，网站速度比以前还快了不少。Cloudflare的配置比较简单，不停点下一步，去域名服务商的网站上修改DNS服务器就搞定了。等待域名解析生效后，跟踪一下路由，发现网站已经定位到Cloudflare的美国机房了。下面是晚上上网高峰期的测速图，以前托管在Github上时，可是黄橙橙红通通一大片，现在大部分地区都是绿油油的，华南地区的速度更好一些：

Github + Cloudflare，放心省事，速度还比以前快，后面真的不折腾了。

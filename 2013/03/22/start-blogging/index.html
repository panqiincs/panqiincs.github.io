我的第一篇文章

发表于 2013-03-22 | 分类于 网站维护 | 评论数：

据说每一个用Octopress写博客的人，第一篇文章必然是关于Octopress的，我应该延续这个传统。

一直不满意各大博客网站的风格，广告多，限制大，字体难看，内容乱。此外，它们都不适合用来写嵌有公式和代码的技术博客。最近，我找到了Octopress。我的博客是用组合的力量搭建好的，完美的Blog = Octopress + Github + Markdown，这正符合Unix操作系统的哲学。

在Linux系统下，我用Vim编辑markdown文件，生成博客后部署到到Github Pages上。只需要很简单的配置，你就可以在指定的位置加上社交帐号主页的链接。至于那些细节问题，都是由Octopress搞定。你还可以用Git对博客源文件做版本管理，推送到Github上作为备份。Octopress官方网站是这么介绍自己的：A blogging framework for hackers，用这种方式写博客真有一种Geek的感觉。

关于写博客的好处，请参考刘未鹏的为什么你应该（从现在开始就）写博客。我写博客的目的主要是记录下自己的生活和学习。Github在国内连接不太通畅，我并不奢望能有多少访客和留言，给自己留下一份纪念就行。

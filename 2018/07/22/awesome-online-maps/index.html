在线地图推荐

发表于 2018-07-22 更新于 2019-11-19 分类于 胡言乱语 Disqus：

我是个地理爱好者，闲来无事时喜欢看各种地图。现在网络上各种类型的地图很多，大名鼎鼎的谷歌地图以及现在常用的导航地图就不介绍了，我主要给大家推荐几个不太知名，但非常有用、有趣的地图。

Relief Map

Relief Map是地形图，用不同的颜色来表示不同的海拔高度（或者海洋深度）。从一个地区的地形图能够大致判断其人口承载能力、交通是否便利以及军事上是否安全，深刻地反映了其“天赋”，所以地形图常用于政治、军事和经济的分析当中。我最喜欢的公众号之一——“地球知识局”中的很多地图就取材于此。

比如通过下面的地形图，你就理解了，为什么很多王朝选择定都关中和洛阳，为什么金兵轻易就攻陷了北宋都城开封，为什么南宋抗金抗蒙古要死守襄阳：

The TRUE SIZE

The TRUE SIZE很有趣，你可以将一个国家或地区拖拽到地图上的任意位置，还可以旋转，如图所示：

很多地图是使用墨卡托投影制作而成的，这种投影会使面积产生变形，越靠近两级，放大比例越大。因此，这种地图无法反映真实的大小，The TRUE SIZE的意义就在于此。

比如上面地图左上角的格陵兰岛，尺寸非常夸张。我们将格陵兰拖到中国旁边，实际大小也不过如此：

在中国和俄罗斯旁边，“小日本”这个称号名副其实，但拖到欧洲比划比划，无论从国土面积还是GDP来看，日本都能称霸一方，难怪想脱亚入欧呢：

还有，智利作为世界上最狭长的国家，到底有多长，看，比黑龙江到海南岛的距离还长：

GLOBELAND30

GLOBELAND30是地表覆盖地图，展示了建筑、耕地、森林、草原、荒漠、湿地等地表类型的分布情况。从地表覆盖地图可以大概了解一个国家的自然条件和人口分布情况。

从下图可以看到，日本和朝鲜半岛大部分被森林覆盖（因为山地太多），可耕地很少，而我们有大片适合耕种的土地。而且中国的气候还拥有雨热同期的特点，虽然住起来不舒适，但更适合农作物生长，得以养活更多的人口。中国人占据了东亚最富饶的一块土地。

很多人可能不太相信，国土面积只有中国三分之一的印度，耕地比中国还多，看完下图你就不觉得奇怪了。南亚次大陆农业条件很好，所以印度才得以养育十几亿的人口：

我们将目光从富饶转向贫瘠。中东地区除了荒漠还是荒漠，只有新月沃地和土耳其有连片的大面积耕地。尼罗河携带肥沃的泥沙，在茫茫沙漠中冲积出了埃及的精华区域，不过这形状太诡异了。想想看，如果他们没有石油，该多么绝望。生活在沙漠地区的人们真不容易！

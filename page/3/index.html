在线地图推荐

发表于 2018-07-22 | 分类于 胡言乱语 | 评论数：

我是个地理爱好者，闲来无事时喜欢看各种地图。现在网络上各种类型的地图很多，大名鼎鼎的谷歌地图以及现在常用的导航地图就不用我介绍了，我主要给大家推荐几个不太知名，但非常有用、有趣的地图。

Relief Map

Relief Map是地形图，用不同的颜色来表示不同的海拔高度（或者海洋深度）。从一个地区的地形图能够大致判断其人口承载能力、交通是否便利以及军事上是否安全，深刻地反映了其“天赋”，所以地形图常用于政治、军事和经济的分析当中。我最喜欢的公众号之一——“地球知识局”中的很多地图就取材于此。

又换了个发行版

发表于 2018-06-17 | 分类于 胡言乱语 | 评论数：

几个月前，一直稳定运行的openSUSE Leap 42.3系统突然无法启动。当时考虑到Leap的下一个版本很快就要发布，不想新版本出来后再重装一次系统，便尝试了openSUSE的滚动发行版Tumbleweed。

滚动发行版的好处在于，系统可以永远保持最新版。但过于激进的版本策略也带来了稳定性方面的风险，运气不好的话，系统可以直接崩溃的。但我用的这几个月，系统从来没有出现过崩溃问题，比之前用过的Ubuntu 14.04 LTS稳定多了。期间也发现过几个明显的bug，都是无关紧要的小问题，在后面的更新中都解决了。更新太频繁是唯一能想到的缺点，平均每周都有一个系统升级，每次的升级文件至少有好几百兆，升级花费时间较长。

需求

我前几年买过一个Kindle电子书阅读器。Kindle是电子墨水屏，阅读效果接近纸质书，能保护视力，而且携带方便，续航能力强。但Kindle屏幕小，阅读文学类书籍还行，对于包含大量公式、图表的专业书就不太适合了。后来，索尼公司推出了一款电子墨水屏阅读器DPT-S1，屏幕大小和A4纸相当，非常适合阅读pdf文档。我对它垂涎已久，无奈那时候还未参加工作，囊中羞涩。2017年4月，它的升级版DPT-RP1问世，比S1更轻薄，更便宜，但依然价格不菲。

C++的const关键字

发表于 2018-01-30 | 分类于 编程语言 | 评论数：

const的作用

C++中的const关键字多才多艺且无处不在，它的语法意义并不高深，用法却变化多端。本文总结了const几乎所有的用法：定义常量、修饰指针和引用、修饰函数的参数和返回值、修饰类成员。

定义常量

关键字const对变量的类型加以限定，将变量定义为一个常量，任何对变量赋值的行为在编译阶段即被编译器阻止。因为const对象一旦创建后，其值不能再改变，所以const对象必须被初始化，初始值可以是任意表达式。还可以利用一个对象初始化另一个对象，它们是不是const都无关紧要。
